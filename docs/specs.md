# Amazon・楽天セール通知サービス - 設計書

## 1. プロジェクト概要

### 1.1 目的
- Amazon・楽天の商品価格を自動で追跡し、大幅値下げ時に通知するサービスを構築する
- 複数のプラットフォーム（Web、Twitter、LINE、Chrome拡張機能）で通知を提供する
- アフィリエイトリンク経由の売上でマネタイズする

### 1.2 目標収益
- 初期: 日500円（月15,000円）
- 半年後: 月30,000円
- 1年後: 月100,000円

### 1.3 予算
- 初期開発: 無料（既存の無料サービスを最大限活用）
- 運用コスト: 月1,000円以内
- 収益が月1,000円に達するまで追加投資を最小限に抑える

## 2. システム構成

### 2.1 全体アーキテクチャ
1. **情報収集システム**: Python スクリプト
2. **Webサイト**: Hugo + GitHub Pages
3. **通知システム**: Twitter Bot、LINE Notify、Chrome拡張機能

### 2.2 システム連携図
```
情報収集システム（Python）
  │
  ├── データ保存（JSON/CSV）
  │     │
  │     ├── Webサイト（Hugo）──→ GitHub Pages ──→ ユーザー
  │     │
  │     ├── Twitter Bot ───────→ Twitter ───────→ ユーザー
  │     │
  │     ├── LINE通知 ─────────→ LINE ─────────→ ユーザー
  │     │
  │     └── API ──────────────→ Chrome拡張 ────→ ユーザー
```

## 3. 各コンポーネントの仕様

### 3.1 情報収集システム（Python）

#### 3.1.1 機能
- Amazon・楽天の商品ページをスクレイピングして価格を取得
- 通常価格と比較して値下げ率を計算
- 設定した閾値（デフォルト20%）以上の値下げを検出して通知
- 商品情報と価格履歴をJSON形式で保存

#### 3.1.2 実装詳細
- **使用技術**: Python 3.x, BeautifulSoup, Requests, Pandas
- **スクレイピング間隔**: 3時間ごと（サーバー負荷と規約に配慮）
- **対象商品管理**: 設定ファイルで管理
- **デプロイ方法**: GitHub ActionsまたはPythonAnywhere

#### 3.1.3 エラーハンドリング
- スクレイピング失敗時のリトライ機能
- 接続エラー時のログ記録
- ブロック検出・回避機能

### 3.2 Webサイト（Hugo）

#### 3.2.1 機能
- セール情報の一覧表示
- カテゴリ別・サイト別の絞り込み
- 通知登録フォーム
- 価格履歴グラフ表示

#### 3.2.2 実装詳細
- **使用技術**: Hugo, GitHub Pages
- **テーマ**: PaperMod（カスタマイズ）
- **コンテンツ更新**: GitHub Actionsで自動化
- **SEO対策**: メタタグ最適化、構造化データ

#### 3.2.3 サイト構成
- ホーム: 最新セール情報
- Amazon特価: Amazon限定のセール情報
- 楽天特価: 楽天限定のセール情報
- カテゴリ別: 家電、ファッション、食品などカテゴリ分け
- 通知登録: Twitter、LINE、Chrome拡張の案内
- サイト紹介: サービス概要説明

### 3.3 Twitter Bot

#### 3.3.1 機能
- セール情報の自動ツイート
- 定期的な宣伝ツイート
- ユーザーからのリプライに対応

#### 3.3.2 実装詳細
- **使用技術**: Python, Tweepy
- **ツイート頻度**: 
  - セール発見時: リアルタイム
  - 定期ツイート: 10時、15時、20時
- **アカウント運用ポリシー**: 
  - スパム判定回避（過度な宣伝を避ける）
  - 有益な情報提供に注力

### 3.4 LINE通知システム

#### 3.4.1 機能
- セール情報をリアルタイムで通知
- カテゴリー別の通知設定
- 値下げ率の閾値設定

#### 3.4.2 実装詳細
- **使用技術**: LINE Notify API
- **通知頻度**: セール発見時
- **通知内容**: 商品名、値下げ率、価格、リンク、画像

### 3.5 Chrome拡張機能

#### 3.5.1 機能
- ブラウザ上での価格通知
- 価格履歴表示
- カスタム通知設定

#### 3.5.2 実装詳細
- **使用技術**: JavaScript, Chrome Extension API
- **データ同期**: APIを介してサーバーと通信
- **プライバシー配慮**: 最小限の権限要求

### 4.3 収益シミュレーション

| 施策 | 数値目標 | 収益計算 |
|------|----------|----------|
| Amazon アフィリエイト | 日5クリック×2%成約率 | 0.1件×平均5,000円×3% = 15円/日 |
| 楽天アフィリエイト | 日7クリック×3%成約率 | 0.21件×平均4,000円×4% = 34円/日 |
| A8.net（ポイントサイト紹介） | 週1件成約 | 1,000円÷7 = 143円/日 |
| Google AdSense | 日300PV | 300PV×0.1円 = 30円/日 |
| **合計** | | **222円/日** + 特価品成約 |

※ 特価品成約は変動が大きいため保守的に見積もり
